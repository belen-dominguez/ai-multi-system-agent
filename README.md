## Sistema multi agente

Consigna: dessarrollar un sistema multi agente en el que un Agente Ordenador clasifique la intencion de la consulta del usuario.
Esta clasificacion activa una ruta condicional que delega la tarea de recuperacion de informacion al Agente RAG especializado correspondinete, para brindar una respuesta contextualizada y fundamentada. Todo el flujo dinamico debe implementarse utilizando LangChain y eestar completamente trazado con Lagfuse.

\*Bonus: Desplegar un Agente evaluador dentro de Langfuse que asigne automaticamente un puntaje de calidad de 1 a 10 a cada respuesta RAG, en funcion de la consulta original y la respuesta final.

Nota: las librerias pueden haber sido reemplazadas
