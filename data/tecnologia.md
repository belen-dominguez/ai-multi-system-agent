# Documentación de Tecnología

## Infraestructura

El sistema corre en servidores AWS en la región us-east-1.
Se usan contenedores Docker para todos los servicios.

## Repositorios

El código fuente vive en GitHub bajo la organización empresa-tech.
La rama principal es main, los deploys se hacen desde ahí.

## Soporte

Para reportar un bug se abre un ticket en Jira con etiqueta "bug".
El tiempo de respuesta para incidentes críticos es de 2 horas.

## Accesos

Los accesos a sistemas se solicitan al equipo de IT via Slack en #it-soporte.
